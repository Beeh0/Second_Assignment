50 Git commands and their uses

  1. git init:Creates a new git repository in the current directory
  2. git clone: Clones an existing git repo from a remote URL
  3. git config: Sets configuration options for git
  4. git branch -a: Lists all local and remote branch
  5. git checkout : Switches to an existing branch
  6. git branch : Creates a new branch
  7. git merge: Merges another branch into the current branch
  8. git remote add: Adds a new remote repo
  9. git pull: Fetches and merges changes from a remote repo
  10. git push: Pushes local changes to a remote repo
  11. git add .: Adda all untracked files in the current dir to the staging area
  12. git status: Shows the status of files in the working dir
  13. git commit -m: Creates a new commit with a message
  14. git log: Shows the history of commits in the repo
  15. git show : Shows the details of a specific commit
  16. git checkout --: Unstages a file from the staging area
  17. git reset HEAD: Unstages and removes a file from the working dir
  18. git stash: Temporarily stores uncommited changes for later
  19. git stash pop: Applies stashed changes back to the working dir
  20. git rebase: Rewrites the commit history by reapplying commits onto another branch
  21. git cherry-pick: Selectively applies a specific commit to the current branch
  22. git tag: Creates a tag at a specific point in the commit history
  23. git fetch: Downloads new chnages from a remoote repo without merging them
  24. git diff: Shows the differenve between files in the working dir
  25. git log --graph: Shows the commit history with a branch diagram
  26. git blame: Shows who last modified each line of a file
  27. git rm: Removes a file from the repo
  28. git reflog: Shows a history of your HEAD pointer movements
  29. git reset [optios]: Moves the HEAD pointer to a specific commit
  30. git revert: Creates a new commit that reverses the chnages of a previous commit
  31. git bisect: Performs a binary search through commits to find a bug
  32. git branch -d: Deletes a local branch
  33. git branch -D: Forcefully deletes a local branch, even if it hasn't been merged
  34. git push origin --delete: Deletes a remote branch
  35. git mergetool: Launches a graphical merge conflict resolution tool
  36. git tag -a -m: Creates an annotated tag with a message
  37. git tag -d: Deletes a tag
  38. git push origin : Pushes a tag to a remote repo
  39. git remote -v: Lists all remote repos
  40. git remote rename : Renames a remote repo
  41. git remote rm : Removes a remote repo
  42. git fetch : Fetches updatess from a remote repo without merging
